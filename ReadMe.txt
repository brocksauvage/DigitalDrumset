INTERACTIVE DRUM SET  
Authors: Denae Douglas, Stephen Fulton, Brock Sauvage, Alec Knutsen								 
Date: May 11, 2016	

Main HTML/PHP Files:

FinalProject_DABS.html - Main html page for displaying the interactive drumset.

Project_3_Record.php - Php file that handles the submit recording event from FinalProject_DABS.html.

Recording_Page.html - HTML page that displays a list of all saved recordings.

Play_Recording.php - Page that handles play record event from Recording_Page.html. Will play a specified recording.

DeleteRecord.php - Page that handles delete record click event from Recording_Page.html. Will delete all checked recordings.
